## Overview

Ecom offers a seamless shopping experience with easy product discovery, secure checkout, and comprehensive order tracking. This guide covers everything from browsing to receiving your purchases.

## Browsing Products

### Homepage

Start your shopping journey from the homepage:

- **Featured Products**: Highlighted items selected by sellers and admins
- **Today's Deals**: Special promotions and discounts
- **Category Banners**: Shop by product category
- **New Arrivals**: Recently added products
- **Best Sellers**: Most popular items
- **Flash Deals**: Limited-time offers

### Product Categories

<Steps>
  <Step title="Navigate to Categories">
    Click "Categories" in the main menu
    
    **Screenshot Description**: Main navigation menu with Categories dropdown showing product categories
  </Step>
  
  <Step title="Select Category">
    Choose a main category to browse
  </Step>
  
  <Step title="Filter by Subcategory">
    Narrow down to specific subcategories
  </Step>
  
  <Step title="Browse Products">
    View all products in the selected category
  </Step>
</Steps>

### Search Function

<Steps>
  <Step title="Use Search Bar">
    Enter product names, keywords, or brands in the search box
  </Step>
  
  <Step title="View Search Results">
    Browse matching products
  </Step>
  
  <Step title="Refine Search">
    Use filters to narrow results:
    - Price range
    - Brand
    - Rating
    - Color
    - Size
    - Availability
  </Step>

<Tip>
Search Tips:
- Use specific product names for best results
- Try different spellings if you can't find what you're looking for

- Browse by category for discovery shopping
</Tip>

### Order Details

Each order shows:

- **Order Code**: Unique identifier
- **Order Date**: When placed
- **Delivery Status**: Current fulfillment stage
- **Payment Status**: Paid/Unpaid
- **Order Items**: Products ordered
- **Seller Information**: For each item
- **Shipping Address**: Delivery location
- **Tracking Number**: If provided by seller
- **Total Amount**: Order value

### Order Statuses

#### Delivery Status

- **Pending**: Order received, awaiting processing
- **Confirmed**: Seller confirmed the order
- **Processing**: Order is being prepared
- **Picked Up**: Package collected by courier
- **On The Way**: Package in transit
- **Delivered**: Order delivered successfully
- **Cancelled**: Order cancelled

#### Payment Status

- **Unpaid**: Payment not yet processed
- **Paid**: Payment received and confirmed

## Returns and Refunds

### Initiating a Return

<Steps>
  <Step title="Contact Seller">
    Use the messaging system to contact the seller about the issue
  </Step>
  
  <Step title="Explain the Problem">
    Describe:
    - What's wrong with the product
    - Photos of damage or defect
    - Reason for return
  </Step>
  
  <Step title="Coordinate Return">
    Work with seller to arrange:
    - Return shipping
    - Refund or replacement
    - Return timeline
  </Step>
</Steps>

### Refund Process

Once seller approves return:

- Refund to original payment method
- Or wallet credit
- Processing time: 3-7 business days
- Depends on payment method

## Best Practices

### Before Purchasing

1. **Read Product Descriptions**: Understand exactly what you're buying
2. **Check Reviews**: See what others say about the product
3. **Verify Seller Ratings**: Buy from reputable sellers
4. **Compare Prices**: Check similar products
5. **Understand Shipping**: Know delivery times and costs
6. **Review Return Policy**: Know your options if there's an issue

### During Checkout

1. **Double-Check Address**: Ensure shipping address is correct
2. **Review Cart**: Verify quantities and items
3. **Save Addresses**: Save frequently used addresses
4. **Use Secure Payment**: Choose secure payment methods
5. **Keep Confirmation**: Save order confirmation emails

### After Purchase

1. **Track Orders**: Monitor delivery progress
2. **Inspect on Delivery**: Check items immediately
3. **Contact Seller Promptly**: Report issues quickly
4. **Leave Reviews**: Help the community
5. **Keep Records**: Save invoices and receipts
