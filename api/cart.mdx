## Get Cart Summary

<api method="GET" endpoint="/api/v2/cart-summary" auth="sanctum">
  Get cart summary with totals
</api>

### Response

<ResponseField name="sub_total" type="string">
  Formatted subtotal price
</ResponseField>

<ResponseField name="tax" type="string">
  Formatted total tax
</ResponseField>

<ResponseField name="shipping_cost" type="string">
  Formatted shipping cost
</ResponseField>

<ResponseField name="discount" type="string">
  Formatted discount amount
</ResponseField>

<ResponseField name="grand_total" type="string">
  Formatted grand total
</ResponseField>

<ResponseField name="grand_total_value" type="number">
  Grand total numeric value
</ResponseField>

<ResponseField name="coupon_code" type="string">
  Applied coupon code
</ResponseField>

<ResponseField name="coupon_applied" type="boolean">
  Whether coupon is applied
</ResponseField>

### Example

```bash
curl -X GET https://your-domain.com/api/v2/cart-summary \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

## Add to Cart

<api method="POST" endpoint="/api/v2/carts/add" auth="sanctum">
  Add a product to cart
</api>

### Request

<ParamField body="id" type="integer" required>
  Product ID
</ParamField>

<ParamField body="variant" type="string">
  Product variant (e.g., "Red-Large")
</ParamField>

<ParamField body="quantity" type="integer" required>
  Quantity to add
</ParamField>

<ParamField body="cost_matrix" type="string">
  Cost verification matrix
</ParamField>

### Response

```json
{
  "result": true,
  "message": "Product added to cart successfully"
}
```

### Example

```bash
curl -X POST https://your-domain.com/api/v2/carts/add \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "id": 123,
    "variant": "Red-Large",
    "quantity": 2
  }'
```

<Note>
The API validates:
- Minimum order quantity
- Stock availability
- Auction products cannot be mixed with regular products
- Digital products are handled specially
</Note>

## Process Cart

<api method="POST" endpoint="/api/v2/carts/process" auth="sanctum">
  Process and validate multiple cart items before checkout
</api>

### Request

<ParamField body="cart_ids" type="string" required>
  Comma-separated cart item IDs (e.g., "1,2,3")
</ParamField>

<ParamField body="cart_quantities" type="string" required>
  Comma-separated quantities matching cart_ids (e.g., "2,1,3")
</ParamField>

### Response

```json
{
  "result": true,
  "message": "Cart updated"
}
```

<Warning>
This endpoint validates:
- Minimum order quantities for each product
- Stock availability for each variant
- Digital product handling
</Warning>
